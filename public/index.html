Help & Instructions

How to use ClusterSEO Pro

  1. Upload CSV/TXT file with keywords and optional search volumes
  2. Or manually input keywords in the Manual Input tab
  3. Adjust clustering sensitivity to control how tightly keywords are grouped
  4. Click "Start Clustering Analysis" to process
  5. View results with SERP overlap, intent detection, and volume data

Keyword Intent Classification

  • Informational: Keywords seeking information (what, how, why, guide, tutorial)
  • Transactional: Keywords showing purchase intent (buy, price, order, shop)
  • Commercial: Keywords comparing options (best, review, compare, vs)
  • Navigational: Keywords looking for specific sites (login, website, official)
